About This Project

The NYC Crime Data Explorer is an interactive project that uses information from NYC Open Data to look at reported crime incidents across New York City. Users can search the dataset by borough or crime type to find specific records and explore patterns in criminal activity. The project features charts that summarize crime trends. This allows users to compare boroughs and examine the distribution of different types of offenses. By organizing and visualizing public crime data, the dashboard helps users understand public safety trends and gain insights into crime in the five boroughs of New York City.
